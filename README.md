# 📄 Official Document Assistant

### AI 公文去函與回文撰寫助手

一個專為「正式公文處理」設計的 AI 工具，協助快速生成標準化、具專業語氣的去函與回文內容。

---

## 🚀 專案介紹

在教育現場與行政工作中，公文撰寫常面臨以下挑戰：

* 撰寫耗時，反覆修改
* 語氣不一致或不夠正式
* 不熟悉標準公文格式
* 回文內容難以精準回應來函重點

**Official Document Assistant** 透過 AI 技術，協助使用者快速產出符合規範的公文內容，提升行政效率與文件品質。

---

## ✨ 核心功能

* 📝 **公文去函生成**
  快速產出正式、條理清晰的發文內容

* 📩 **公文回文撰寫**
  根據來函重點，自動整理並回覆重點事項

* 🎯 **語氣專業優化**
  符合政府與教育機構常用公文語氣

* 📚 **結構自動整理**
  內文條列清楚，邏輯完整

* ⚡ **提升行政效率**
  大幅縮短撰寫與修改時間

---

## 🎯 適用對象

* 🏫 學校行政人員（幼兒園、國中小、高中）
* 🏢 政府機關承辦人
* 📄 教育單位與非營利組織
* 👩‍💼 需處理正式往來文件之行政人員

### 2️⃣ AI 自動產出內容

#### 📤 去函範例

```markdown
主旨：有關本園辦理親師座談會一案，請查照。

說明：
一、為增進親師合作，促進幼兒學習發展，特辦理本次座談會。
二、活動時間：114年3月12日（星期三）下午7時。
三、活動地點：本園多功能教室。
四、敬請貴家長踴躍參與，共同關心幼兒學習歷程。

正本：本園全體家長
副本：本園行政單位
```

---

#### 📩 回文範例

```markdown
主旨：有關貴家長詢問親師座談會時間調整一案，復如說明，請查照。

說明：
一、經評估整體活動安排與場地使用情形，本次座談會時間維持原訂規劃。
二、如家長當日不克出席，將另提供會議重點摘要，以利了解相關內容。

正本：○○家長
副本：本園行政單位
```
