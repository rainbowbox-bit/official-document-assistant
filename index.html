如何取得 Gemini API 金鑰？

          
    
            
  1. 
              前往 Google AI Studio
    
              開啟瀏覽器，前往
              aistudio.google.com ↗
            
    2. 
            
  2. 
              登入 Google 帳號
    
              使用個人或機構 Google 帳號登入（建議使用個人帳號以避免機構政策限制）
            
    3. 
            
  3. 
              建立 API 金鑰
    
              點擊左側「Get API key」→「Create API key」→ 選擇或建立一個 Google Cloud 專案
            
    4. 
            
  4. 
              複製並儲存
    
              複製以「AIza」開頭的金鑰，貼至上方欄位後按「儲存金鑰」
            
    5. 
          


          

            🔒
            

              安全說明：API 金鑰僅儲存於您瀏覽器的 localStorage，
              不會傳送至任何中間伺服器。所有 API 請求直接從您的瀏覽器發至 Google Gemini。
            

          

          

            💡
